# 🎮 Number Guessing Game (Java)

A simple command-line game built in Java where the computer picks a 
random number and you have to guess it!

## 🎯 How to Play

1. Run the program
2. Enter a number between 1 and 100
3. The program will tell you if your guess is **Too High** or **Too Low**
4. Keep guessing until you find the correct number
5. Your total attempts will be shown at the end

## 📸 Example
Guess a Number Between 1-100
50
Your Guess is Too High
25
Your Guess is Too Low
37
Your Guess is Too High
31
You got it in 4 attempts! 🎉

## 🛠️ Built With

- Java
- IntelliJ IDEA

## 🚀 How to Run

1. Make sure Java is installed on your machine
2. Clone this repository
3. Open `Main.java` in IntelliJ IDEA
4. Hit the Run button ▶️
